### 🗺️ [Trip Calculator](https://www.tripcalculator.in) — Full-Stack Money Spend Traking Management 
> *Project Description*

Trip Calculator is a full-stack travel expense management platform that helps users plan budgets, track spending, and manage trip-related expenses efficiently. The application provides secure authentication, expense categorization, spending analytics, and responsive access across devices, making travel budgeting simple and organized.

**Tech Stack:** React.js • Tailwind CSS • Node.js • Express • MongoDB  • JWT • SendGrid

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-TourGuy-4ea94b?style=for-the-badge)](https://tripcalculator.in)

## Key Features

- 💰 Track trip expenses in real time
- 📊 Budget planning and spending analysis
- 🧾 Categorize expenses (Food, Transport, Hotel, Shopping, etc.)
- 📈 Expense summary with detailed breakdown
- 👥 Group trip expense management
- 🔐 JWT-based authentication and secure user sessions
- 📧 Email notifications and account verification using SendGrid
- ☁️ MongoDB-powered data storage and management
- 📱 Fully responsive UI for mobile, tablet, and desktop
- ⚡ Fast and modern React + Tailwind CSS interface
- 🌍 Manage multiple trips from a single dashboard
- 🔄 Real-time expense updates and tracking
- 🛡️ Protected routes and role-based access control

### 🗺️ [Neon Air Draw — AI-Powered Spatial Drawing Interface](https://air-green-one.vercel.app/) — Full-Stack Money Spend TRaking Management 
> *Project Overview*

Neon Air Draw is a real-time AI-powered web application that enables users to draw, erase, and manipulate digital sketches using hand gestures in mid-air. Leveraging computer vision and hand-tracking technology, the system transforms natural hand movements into an intuitive spatial drawing experience inspired by futuristic interfaces.
**Tech Stack:** 
Frontend: React.js, Vite
Hand Tracking: MediaPipe Hands
Animations: Framer Motion
Graphics: WebGL
Icons: Lucide React
Styling: Modern CSS, Glassmorphism UI

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-TourGuy-4ea94b?style=for-the-badge)](https://air-green-one.vercel.app/)

## Key Features

- ✋ Real-time hand gesture recognition using AI-powered tracking
- 🎨 Draw and erase sketches in the air without physical input devices
- 🖐️ Dual-hand interaction system for drawing and object control
- 📐 Non-destructive transformations including move, scale, and rotate
- ⚡ WebGL-accelerated rendering for smooth, high-performance interactions
- 🌀 Snap-to-angle rotation and inertia-based object movement
- 🕶️ Modern glassmorphism interface with neon visual effects
- 📱 Responsive and interactive user experience
- 📖 Integrated gesture guide and onboarding system

## Gesture Controls

### ✍️ Drawing Hand (Right Hand)

- ☝️ Index Finger Up — Draw
- 🤏 Pinch — Erase
- ✊ Fist — Clear Canvas

### 🖐️ Control Hand (Left Hand)

- ✌️ Two Fingers — Move Objects
- 🤏 Pinch & Spread — Scale Objects
- 🤚 Open Palm — Rotate Objects
